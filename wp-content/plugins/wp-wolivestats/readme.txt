=== Wolive Real Time Analytics ===
Requires at least: 4.6
Tested up to: 4.7
Stable tag: 4.6


== Description ==

Wolive is a plugin to allows track every single visitor of your wordpress & woocommerce site in real time. 

Track actions and all their activiy in 100% Real Time.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-wolivestats` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
